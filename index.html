Neural-Symbolic CQA (In progress)
Academic Research
12/2018
Proposed a neural-symbolic complex question answering (NS-CQA) system, which contains (a) a neural
"generator", a sequence-to-sequence model that transforms the natural language query into primitive action
sequences and (b) a symbolic "executor" that performs discrete executions. By employing REINFORCE
method, the task reward of the structured executions can be directly optimized.
Knowledge Base Relation Detection via Memory Networks (In progress)
Academic Research
12/2018
Proposed a relation detection model based on memory networks. Different from previous methods, when
we calculate the similarity between given question Q and a given relation R, we also take the similarity
between Q and the questions whose corresponding relation is R into account.
